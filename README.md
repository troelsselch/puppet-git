Puppet module for Git
=====================

This class installs the git-core package and a global gitconfig file.

Fork from https://github.com/wulff/puppet-git. Added support for CentOS.

Support
-------

Tested on CentOS 6.4 

Basic usage
-----------

Install Git:

    class { 'git': }
